# Helpers Plugin

The **helpers** plugin provides a collection of **small functional CSS classes** that can be used for creating or customizing components. The main aim of this plugin is to provide a collection of reusable classes to replace the conventional style of writting CSS classes.

To load the helpers plugin, import the **plugins module** and call the `plugins.helpers()` function in the `$plugins` section of your configuration:

```scss
@use "siimple/plugins" as plugins;

@use "siimple" with (
    $plugins: (
        plugins.helpers(),
    ),
);
```

Additionally, in case that you are only interested in some helpers you can load each helper individually instead of loading all helpers:

```scss
@use "siimple/plugins" as plugins;

@use "siimple" with (
    $plugins: (
        plugins.color(),
        plugins.spacing(),
        plugins.sizing(),
    ),
));
```


### Usage

Let's see how this plugin works with a simple example. In a tradditional way, for creating a new component we should write the CSS styles in a `<style>` tag or in a separate CSS file:

```html
<style>
    .card {
        background-color: #ffffff;
        border-radius: 0.75rem;
        display: flex;
        margin-left: auto;
        margin-right: auto;
        max-width: 30rem;
        padding: 1.5rem;
    }
    .card-image {
        flex-shrink: 0;
        margin-right: 1.5rem;
    }
    .card-image img {
        border-radius: 50%;
        height: 6rem;
        width: 6rem;
    }
    .card-content {
        flex-grow: 1;
    }
    .card-content-title {
        font-size: 2rem;
        font-weight: bold;
        margin-bottom: 0.5rem;
    }
    .card-content-description {
        color: rgba(116,124,139, 1);
    }
</style>

<div class="card">
    <div class="card-image">
        <img src="/landscape1.jpg" />
    </div>
    <div class="card-content">
        <div class="card-content-title">John Doe</div>
        <div class="card-content-description">
            Lorem ipsum dolor sit amet . . .
        </div>
    </div>
</div>
```

But with the **helpers** module, you can just build this component using only HTML, without writting a single line of CSS:

```html live=true bg=blue-200
<div class="has-bg-white has-radius-lg has-d-flex has-maxw-120 has-mx-auto has-p-6">
    <div class="has-flex-no-shrink has-mr-6">
        <img class="has-radius-full has-h-24 has-w-24" src="/landscape1.jpg" />
    </div>
    <div class="has-flex-grow">
        <div class="has-mb-2 has-weight-bold has-text-2xl">John Doe</div>
        <div class="has-text-coolgray-500">
            Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod 
            tempor incididunt ut labore et dolore magna aliqua.
        </div>
    </div>
</div>
```


### Building new helpers

You can generate your own helpers using the `create-helper-plugin` function. This function accepts only a map as an argument, where the configuration of the helper must be provided:

```scss
// my-helpers.scss
@use "siimple/lib" as lib;

@function my-helper() {
    @return lib.create-helper-plugin((
        // helper configuration here
    ));
}
```

The configuration accepts the following fields:

- `name`: name of the new helper.
- `properties`: a list of CSS properties to set in the helper.
- `values`: a list of values to set in each CSS property specified in the `properties` field.
- `responsive`: a boolean to generate responsive variants of this helper.
- `states`: a list of states to generate variants. Allowed values: `default`, `hover` and `focus`.

The following code shows a simple example of a custom helper that will generate helpers to set the `background-color` property:

```scss
// my-helpers.scss
@use "siimple/lib" as lib;

@function background-colors() {
    @return lib.create-helper-plugin((
        "name": "bg",
        "properties": ("background-color"),
        "states": ("default" "hover"),
        "responsive": false,
        "values": (
            "transparent": transparent,
            "black": #000000,
            "white": #ffffff,
        ),
    ));
}
```

Now you can add your new helper plugin to the `$plugins` section of the siimple configuration:

```scss
@use "siimple/plugins" as plugins;
@use "./my-helpers.scss" as my-helpers;

@use "siimple" with (
    $plugins: (
        my-helpers.background-colors(),
    ),
));
```
