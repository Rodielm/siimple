# Crumb

The breadcrumb is an easy-to-use element to keep track of the path the user has navigated. 
Each item should have the `siimple-crumb` class, and you can use the additional `siimple-crumbs` class to wrap all crumb elements:

```html live=true
<div class="siimple-crumbs has-mb-0">
    <a href="#" class="siimple-crumb">Home</a>
    <a href="#" class="siimple-crumb">About</a>
    <a href="#" class="siimple-crumb">Contact</a>
</div>
```

We provide an `active` variant to mark a crumb as active, for example to inform the user about the current page.

```html live=true
<div class="siimple-crumbs has-mb-0">
    <a href="#" class="siimple-crumb">Home</a>
    <a href="#" class="siimple-crumb">About</a>
    <a href="#" class="siimple-crumb is-active">Contact us</a>
</div>
```

### Variants

Variants for this component can be defined in the `navigation.crumb` section of the theme configuration:

```scss
siimple.theme((
    "navigation.crumb": (
        "active": (
            "cursor": "default",
            "opacity": "1.0",
            "pointer-events": "none",
        ),
    ),
));
```
