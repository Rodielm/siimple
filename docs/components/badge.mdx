# Badge

**Badges** are useful elements for indicating status, attach metadata to another element or displaying numbers for counts. You can easily turn a `<span>` or `<div>` element to a badge adding the `siimple-badge` class.

```html live=true
<span class="siimple-badge">Badge</span>
<span class="siimple-badge is-secondary">Badge</span>
```

### Variants

Variants for this component can be defined in the `badge` section of the theme configuration:

```scss
siimple.theme((
    "badge": (
        "primary": (
            "background-color": "primary",
            "color": "white",
        ),
        "secondary": (
            "background-color": "secondary",
            "color": "white",
        ),
    ),
));
```
