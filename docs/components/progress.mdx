# Progress

> This component is part of the **experimental** components module.

Style your `<progress>` elements using the class `siimple-progress`:

```html live=true
<progress class="siimple-progress" max="1" value="0.5"></progress>
```

### Variants

Variants for this component can be defined in the `form.progress` section of the theme configuration:

```scss
siimple.theme((
    "form.progress": (
        "default": (
            "background-color": "fill",
            "color": "primary",
        ),
    ),
));
```
