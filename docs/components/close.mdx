# Close

Use the `siimple-close` class to create a close button:

```html align=center live=true
<div class="siimple-close"></div>
```


### Variants

Variants for this component can be defined in the `button.close` section of the theme configuration:

```scss
siimple.theme((
    "button.close": (
        "default": (
            "color": "currentColor",
        ),
    ),
));
```
