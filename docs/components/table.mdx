# Table

The `siimple-table` class allows you to style your `<table>` elements. We provide some variants to the default table style:

- `divided`: display a divider rule between table rows.
- `bordered`: display a table with borders.
- `striped`: table will have a zebra striped style.
- `hoverable`: highlight the row where the cursor is.
- `fixed`: automatically divide equally the width of the columns across the table, regardless of content inside the cells.


```html live=true
<table class="siimple-table is-divided is-hoverable">
    <thead>
        <tr>
            <th>Header 1</th>
            <th>Header 2</th>
            <th>Header 3</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Cell 1</td>
            <td>Cell 2</td>
            <td>Cell 3</td>
        </tr>
        <tr>
            <td>Cell 4</td>
            <td>Cell 5</td>
            <td>Cell 6</td>
        </tr>
    </tbody>
</table>
```

You can also use the special `siimple-table-responsive` class to allow creating responsive tables.

```html
<div class="siimple-table-responsive">
    <table class="siimple-table">
        . . .
    </table>
</div>
```

### Variants

Variants for this component can be defined in the `table` section of the theme configuration:

```scss
siimple.theme((
    "table": (
        "default": (
            "& th, & td": (
                "background-color": "white",
                "padding": "1rem",
            ),
        ),
        "divided": (
            "& thead th": (
                "border-bottom-color": "fill",
                "border-bottom-style": "solid",
                "border-bottom-width": "0.125rem",
            ),
            "& tbody td": (
                "border-top-color": "fill",
                "border-top-style": "solid",
                "border-top-width": "0.125rem",
            ),
        ),
    ),
));
```
