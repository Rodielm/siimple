# Background

### Background color

Use the following classes to set the background color of any element: 

> This helper will extract the values from the **backgrounds** section of the theme. You may have extra or different classes based on your theme configuration.


| Class | Color | |
|-------|-------|-|
| `has-bg-primary`   | `#307ad5`      | <div class="has-bg-primary   has-radius-full has-w-6 has-h-6"></div> |
| `has-bg-secondary` | `#38c7a3`      | <div class="has-bg-secondary has-radius-full has-w-6 has-h-6"></div> |
| `has-bg-highlight` | `#e5effb`      | <div class="has-bg-highlight has-radius-full has-w-6 has-h-6"></div> | 
| `has-bg-fill`      | `#ebedef`      | <div class="has-bg-fill      has-radius-full has-w-6 has-h-6"></div> | 
| `has-bg-muted`     | `#747c8b`      | <div class="has-bg-muted     has-radius-full has-w-6 has-h-6"></div> | 
| `has-bg-dark`      | `#363f4f`      | <div class="has-bg-dark      has-radius-full has-w-6 has-h-6"></div> | 
| `has-bg-light`     | `#ebedef`      | <div class="has-bg-light     has-radius-full has-w-6 has-h-6"></div> | 
| `has-bg-white`     | `#ffffff`      | <div class="has-bg-white     has-radius-full has-w-6 has-h-6"></div> | 
| `has-bg-black`     | `#000000`      | <div class="has-bg-black     has-radius-full has-w-6 has-h-6"></div> |
| `has-bg-current`   | `currentColor` | <div class="has-bg-current   has-radius-full has-w-6 has-h-6"></div> |

Remember that background color helpers **do not set the text color attribute**, so you should use one of the `has-text-{color}` color helpers.


#### Background color on hover

You can change the background color of any element on hover adding the `hover:` prefix to the background color utility.

```html live=true bg=blue-200 color=blue-700 align=center
<div class="siimple-btn hover:has-bg-primary has-bg-secondary has-text-white">
    <strong>Hover me</strong>
</div>
```


### Background repeat 

Control the background image repeat using the following classes:

| Class | Property |
|-------|----------|
| `has-bg-repeat` | `background-repeat: repeat;` |
| `has-bg-repeat-x` | `background-repeat: repeat-x;` |
| `has-bg-repeat-y` | `background-repeat: repeat-y;` |
| `has-bg-no-repeat` | `background-repeat: no-repeat;` |


### Background size

Manage the size of the background image using the following classes:

| Class | Property |
|-------|----------|
| `has-bg-auto` | `background-size: auto;` |
| `has-bg-cover` | `background-size: cover;` |
| `has-bg-contain` | `background-size: contain;` |


### Disabling background helpers

If you do not need the background helpers in your project, you can disable them by adding to the `exclude` list in the `helpers` plugin configuration:

```scss
@use "siimple/plugins" as plugins;

@use "siimple" with (
    $plugins: (
        plugins.helpers((
            "exclude": (
                "background.color",
                "background.repeat",
                "background.size",
            ),
        )),
    ),
);
```
