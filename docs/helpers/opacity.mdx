# Opacity

Use the following helpers classes to control the opacity of an element:

> This helper will extract the values from the **opacities** section of the theme. You may have extra or different classes based on your theme configuration.


| Class | Property |
|-------|----------|
| `has-opacity-0` | `opacity: 0.00;` |
| `has-opacity-5` | `opacity: 0.05;` |
| `has-opacity-10` | `opacity: 0.10;` |
| `has-opacity-15` | `opacity: 0.15;` |
| `has-opacity-20` | `opacity: 0.20;` |
| `has-opacity-25` | `opacity: 0.25;` |
| `has-opacity-30` | `opacity: 0.30;` |
| `has-opacity-35` | `opacity: 0.35;` |
| `has-opacity-40` | `opacity: 0.40;` |
| `has-opacity-45` | `opacity: 0.45;` |
| `has-opacity-50` | `opacity: 0.50;` |
| `has-opacity-55` | `opacity: 0.55;` |
| `has-opacity-60` | `opacity: 0.60;` |
| `has-opacity-65` | `opacity: 0.65;` |
| `has-opacity-70` | `opacity: 0.70;` |
| `has-opacity-75` | `opacity: 0.75;` |
| `has-opacity-80` | `opacity: 0.80;` |
| `has-opacity-85` | `opacity: 0.85;` |
| `has-opacity-90` | `opacity: 0.90;` |
| `has-opacity-95` | `opacity: 0.95;` |
| `has-opacity-100` | `opacity: 1.00;` |


```html live=true bg=blue-200 color=white

```

### Opacity on hover

You can change the opacity of any element on hover adding the `hover:` prefix to the opacity helper.

```html
<div class="has-opacity-50 hover:has-opacity-100 ...">
    . . .
</div>
```

### Responsive opacity

Control the opacity of an element at a specific breakpoint adding a `{breakpoint}:` prefix to the opacity class.

```html
<div class="has-opacity-50 mobile:has-opacity-100 ...">
    . . . 
</div>
```

### Disabling opacity helpers

If you do not need the opacity helpers in your project, you can disable them by adding to the `exclude` list in the `helpers` plugin configuration:

```scss
@use "siimple/plugins" as plugins;

@use "siimple" with (
    $plugins: (
        plugins.helpers((
            "exclude": (
                "opacity",
            ),
        ),
    ),
);
```
