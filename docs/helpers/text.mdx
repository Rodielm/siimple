# Text

### Text alignment

Use one of the following classes to set the horizontal align of your text inside the container element:

| Class | Property |
|-------|----------|
| `has-text-justify` | `text-align: justify;` |
| `has-text-left` | `text-align: left;` |
| `has-text-center` | `text-align: center;` |
| `has-text-right` | `text-align: right;` |

Use the `has-text-justify` helper to justify the inline contents:

```html live=true bg=blue-200 color=blue-700
<div class="has-text-justify">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor 
    incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud 
    exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.
</div>
```

Use the `has-text-left` helper to align inline contents to the left: 

```html live=true bg=blue-200 color=blue-700
<div class="has-text-left">Left aligned text</div>
```

Use the `has-text-center` helper to center inline contents:

```html live=true bg=blue-200 color=blue-700
<div class="has-text-center">Centered text</div>
```

Use the `has-text-right` helper to align inline contents to the right:

```html live=true bg=blue-200 color=blue-700
<div class="has-text-right">Right aligned text</div>
```


### Text transform

You can use these classes to apply text transformations: 

| Class | Property |
|-------|----------|
| `has-text-uppercase` | `text-transform: uppercase;` |
| `has-text-lowercase` | `text-transform: lowercase;` |
| `has-text-capitalize` | `text-transform: capitalize;` |


Use the `has-text-uppercase` helper to convert all characters of your text to uppercase:

```html live=true bg=blue-200 color=blue-700
<strong class="has-text-uppercase">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit...
</strong>
```

Use the `has-text-lowercase` helper to convert all characters of your text to lowercase:

```html live=true bg=blue-200 color=blue-700
<strong class="has-text-lowercase">
    Lorem Ipsum Dolor Sit Amet, Consectetur Adipiscing Elit...
</strong>
```

Use the `has-text-capitalize` helper to convert the first character of each word of your text to uppercase:

```html live=true bg=blue-200 color=blue-700
<strong class="has-text-capitalize">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit...
</strong>
```

### Text decoration

Use the following classes to set the `text-decoration` property of your text:

| Class | Property |
|-------|----------|
| `has-text-underline` | `text-decoration: underline;` |
| `has-text-no-underline` | `text-decoration: none;` |


Use the `has-text-underline` to underline the text:

```html live=true bg=blue-200 color=blue-700
<strong class="has-text-underline">
    Lorem ipsum dolor sit amet, consectetur adipiscing elit...
</strong>
```

### Text color

Use `has-text-{color}` to apply a different color to the text. 

> This helper will extract the values from the **colors** section of the theme. You may have extra or different classes based on your theme configuration.

| Class | Property | |
|-------|----------|-|
| `has-text-primary` | `color: #307ad5;` | <strong class="has-text-primary">Aa</strong> |
| `has-text-secondary` | `color: #38c7a3;` | <strong class="has-text-secondary">Aa</strong> |
| `has-text-body` | `color: #101623;` | <strong class="has-text-body">Aa</strong> | 
| `has-text-heading` | `color: #101623;` | <strong class="has-text-heading">Aa</strong> | 
| `has-text-muted` | `color: #747c8b;` | <strong class="has-text-muted">Aa</strong> | 


##### Text color on hover

You can change the text color of any element on hover adding the `hover:` prefix to the text color utility.

```html live=true bg=blue-200 align=center
<strong class="has-text-muted hover:has-text-primary">Hover me</strong>
```

### Disabling text helpers

If you do not need the text helpers in your project, you can disable them by adding to the `exclude` list in the `helpers` plugin configuration:

```scss
@use "siimple/plugins" as plugins;

@use "siimple" with (
    $plugins: (
        plugins.helpers((
            "exclude": (
                "text.align",
                "text.color",
                "text.decoration",
                "text.overflow",
            ),
        ),
    ),
);
```
