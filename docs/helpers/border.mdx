# Border

### Border radius

Use the following classes to control the border radius of any element:

> This helper will extract the values from the **radius** section of the theme. You may have extra or different classes based on your theme configuration.


| Class | Property |
|-------|----------|
| `has-radius-normal` | `border-radius: 0.5rem;` |
| `has-radius-small` | `border-radius: 0.25rem;` |
| `has-radius-large` | `border-radius: 1rem;` |
| `has-radius-full` | `border-radius: 9999px;` |
| `has-radius-none` | `border-radius: 0px;` |


You can use these classes to apply different border radius to your elements:

```html live=true centered=true bg=blue-200 color=white
<div class="has-p-6 has-m-2 has-bg-primary has-radius-small">
    <strong>.has-radius-small</strong>
</div>
<div class="has-p-6 has-m-2 has-bg-primary has-radius-normal">
    <strong>.has-radius-normal</strong>
</div>
<div class="has-p-6 has-m-2 has-bg-primary has-radius-large">
    <strong>.has-radius-large</strong>
</div>
```

You can use the `has-radius-full` class to create pills or circles:

```html live=true centered=true bg=blue-200 color=white
<div class="has-px-6 has-py-3 has-m-2 has-bg-primary has-radius-full">
    <strong>Pill element</strong>
</div>
<div class="has-w-24 has-h-24 has-m-2 has-bg-primary has-flex has-items-center has-justify-center has-radius-full">
    <strong>Circle</strong>
</div>
```

You can remove the border radius using `has-radius-none`: 

```html live=true centered=true bg=blue-200 color=white
<div class="has-p-6 has-m-2 has-bg-primary has-radius-none">
    <strong>.has-radius-none</strong>
</div>
```




### Disabling border helpers

If you do not need the border helpers in your project, you can disable them by adding to the `exclude` list in the `helpers` plugin configuration:

```scss
@use "siimple/plugins" as plugins;

@use "siimple" with (
    $plugins: (
        plugins.helpers((
            "exclude": (
                "border.radius",
            ),
        ),
    ),
);
```
